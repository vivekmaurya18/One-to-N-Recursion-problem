# 🔢 1 to N Print using Recursion in Java

## Introduction
This program prints numbers from **1 to N** using the concept of **recursion**.

👉 Recursion ka matlab hai: function khud ko call karta hai jab tak base condition satisfy na ho jaye.

---

## Problem
Given a number `N`, print numbers from **1 to N**.

📌 Example:
Input: `5`  
Output: `1 2 3 4 5`


## Logic (Simple Explanation)
- Jab tak `n == 0` nahi hota → function call hota rahega
- Jab `n == 0` ho jata hai → function stop ho jata hai
- Print **baad me hota hai** (return phase me)

## Complexity
- Time: O(N)
- Space: O(N) (stack space)
